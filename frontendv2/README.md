# Pylon Frontend v2

Professional landing page and user dashboard for the Pylon platform.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server (port 3001)
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3001](http://localhost:3001) to view the landing page.

Open [http://localhost:3001/user](http://localhost:3001/user) to view the user dashboard.

## Pages

- **/** - Landing page with hero, features, and stats
- **/platform** - Technical architecture and agents
- **/solution** - Problem statement and agent workflow
- **/industries** - Industry use cases and business model
- **/capabilities** - Platform capabilities and data sources
- **/learn** - Documentation and team info
- **/user** - User dashboard (main application interface)

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Design System

- **Colors**: Dark navy (#0a0e1a), White, Green accent (#10b981)
- **Typography**: Inter font, large headlines, generous spacing
- **Layout**: Clean Palantir-inspired aesthetic with lots of whitespace
